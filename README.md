# finance-manager
Консольное приложение для управления личными финансами на Python.

## 📋 Возможности

- ✅ Добавление доходов и расходов
- ✅ Автоматическое сохранение данных в JSON-файл
- ✅ Просмотр текущего баланса
- ✅ Фильтрация транзакций по категориям
- ✅ Фильтрация по диапазону дат
- ✅ Статистика по категориям

## 🚀 Установка и запуск

1. Клонируйте репозиторий:
```bash
git clone https://github.com/ВАШ_АККАУНТ/personal-finance-manager.git
cd personal-finance-manager
```
2.Убедитесь, что у вас установлен Python 3.7 или выше:

```bash
python --version
```
3.Запустите программу:

```bash
python main.py
```
## 📁 Структура проекта
```text
personal-finance-manager/
├── main.py          # Главный файл для запуска
├── models.py        # Классы Transaction и FinanceManager
├── storage.py       # Работа с файловым хранилищем
├── utils.py         # Вспомогательные функции
├── data/            # Папка с данными (создаётся автоматически)
│   └── transactions.json
└── README.md
```
